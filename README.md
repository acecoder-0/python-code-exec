# Dockerized Python Code Execution Web Application

A Flask-based web application that allows users to write and execute Python code through a browser interface. The application is containerized using Docker and can be deployed locally or shared through Docker Hub.

## Features
- Web-based Python code editor
- Real-time code execution
- Output display in browser
- Dockerized deployment
- Easy sharing using Docker Hub

## Tech Stack
- Python
- Flask
- Docker
- GitHub
- Docker Hub

## Run Locally

### Build Image
```bash
docker build -t python-code-exec .
